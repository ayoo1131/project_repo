About This Site

Overview

This website is a Full Stack Web Development portfolio created by me, Andrew Yoo, to showcase the applications I've designed and built. The site has a user registration ability for users to create an account and save information for future use. Currently there are 2 production applications ready for use.

Appliations

Applied Jobs

Overview: This application helps job seekers track the status of their job applications. It let's users organize job applications by status(applied, denied, interview) and gives users data on how long it's been since they've applied.

Features:

• Save job application with: company, job title, date applied, location, and URL

• Display job applications by application status (applied, rejected, interviews)

• Sort job applications by date, name, etc.

• Update the appliation status of a job application

Cover Letter

Overview: This application lets the user store their contact information to quickly create a framework of a cover letter. The user can write the body of the cover letter and download it to their personal device as a word document.

Features:

• Save applicant info: name, email, phone no., social media(optional), extra(optional)

• Update and delete applicant info

• Fill cover letter template with applicant info and cover letter body paragraphs

• Download cover letter as a Word document
